🩸 Pranjyoti Blood Donation & Donor Management System
📌 Project Overview

The Pranjyoti Blood Donation & Donor Management System is a web-based application designed to manage blood donors, recipients, and blood stock efficiently. The system helps blood banks and organizations streamline donor registration, track blood availability, and manage donation records.

🎯 Objectives
Maintain a centralized database of blood donors
Manage blood stock levels
Facilitate quick search for donors based on blood group and location
Record and track blood donations
Improve communication between donors and recipients
🚀 Features
👤 Donor Module
Donor registration
Update donor details
View donation history
Search donors by blood group
🏥 Admin Module
Add / update / delete donor records
Manage blood stock
View all registered donors
Monitor donation activities
🔍 Search Functionality
Search donors by:
Blood group
City / Location
📊 Blood Management
Track available blood units
Update stock after donation or usage
🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: PHP
Database: MySQL
Server: XAMPP / Apache

Install XAMPP or any local server

Copy project folder to:

C:\xampp\htdocs\
Start Apache and MySQL
Import database:
Open phpMyAdmin
Create a database (e.g., blood_db)
Import .sql file
